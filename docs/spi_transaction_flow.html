Key conditions
IDLE
Precondition
o_busy = 0 — Must verify this before triggering transfer · state = STATE_IDLE · o_ss = HIGH · o_sclk = LOW · o_mosi = Z
Slave preload
i_tx_ready = 1 → r_tx_byte ← i_tx_byte (only accepted when r_ss = 1) · Must happen BEFORE master fires tx_ready
Master trigger
i_tx_ready = 1 → r_tx_byte ← i_tx_byte → next_state = STATE_PRE_COMM · o_busy ← 1
i_tx_ready pulse → FSM transitions to PRE_COMM
PRE_COMM
SS asserted
r_ss ← 0 → o_ss = LOW (slave now selected) · o_mosi drives (was Z, now active)
First bit loaded
w_tx_byte = r_tx_byte << 0 → r_mosi ← w_tx_byte[7] (MSB of tx byte) · r_cycle_count ← 1 · o_mosi = bit[7]
Slave detects SS
r_ss ← 0 (registered i_ss goes low) → o_busy = 1 · w_tx_byte[7] → o_miso drives bit[7] of slave tx
SCLK still idle
o_sclk = LOW — clock hasn't started yet · r_cont_sclk = 0 · Both MOSI and MISO are stable with bit[7] before first clock edge
FSM auto-transitions to COMM after 1 cycle
COMM
SCLK generation
r_cont_sclk counts 0 → DIVIDE_FREQ=1 → toggle r_sclk → o_sclk toggles · r_edge_detect = 1 for 1 cycle
MOSI shift out
On SCLK falling edge (r_sclk=0 && r_edge_detect): w_tx_byte = r_tx_byte << r_cycle_count → r_mosi ← w_tx_byte[7] → o_mosi = next bit · r_cycle_count++
MISO sample in
On SCLK rising edge (o_sclk=1 && r_edge_detect_tmp): r_rx_byte ← {r_rx_byte[6:0], i_miso} — shift in 1 bit from slave · r_edge_detect_tmp is r_edge_detect delayed 1 cycle (metastability guard)
Slave MISO drive
On SCLK falling edge (negedge w_sclk, Mode 0): r_cycle_count++ → w_tx_byte = r_tx_byte << r_cycle_count → w_miso = w_tx_byte[7] → o_miso updates
Slave MOSI sample
On SCLK rising edge (posedge w_sclk, Mode 0): r_rx_byte ← {r_rx_byte[6:0], r_mosi} — shift in 1 bit from master · Uses registered r_mosi (1 FPGA clock delayed for CDC safety)
Exit condition
r_cycle_count ≥ 8 AND r_edge_detect = 1 → next_state = STATE_POS_COMM — all 8 bits transferred
r_cycle_count = 8 → FSM transitions to POS_COMM
POS_COMM
SCLK goes idle
r_sclk ← 0 → o_sclk = LOW (CPOL=0 idle level) · r_edge_detect = 0 · Slave has time to latch last SCLK edge
Hold timer
r_count_pos_com counts up each FPGA clock → exit when = DIVIDE_FREQ >> 1 = 0 (immediate for DIVIDE_FREQ=1)
SS deasserted
r_ss ← 1 → o_ss = HIGH (slave deselected) · o_mosi = Z (tri-state, releases bus) · o_miso = Z (slave releases MISO)
r_count_pos_com reaches limit → FSM returns to IDLE
OUTPUT
Master output
state=IDLE AND o_busy was 1 (transition): o_rx_byte ← r_rx_byte · o_byte_ready = 1 for exactly 1 FPGA clock · o_busy ← 0
Slave output
r_ss=1 AND o_busy was 1 (SS just rose): o_rx_byte ← r_rx_byte · o_byte_ready = 1 for exactly 1 FPGA clock · o_busy ← 0
UVM scoreboard
master_monitor sees o_byte_ready=1 → captures o_rx_byte → sends to scoreboard · slave_monitor sees o_byte_ready=1 → captures o_rx_byte → sends to scoreboard · check: slave_rx == master_tx AND master_rx == slave_tx
📊 Signal Waveform — Sending 0x55 (Master) ↔ 0xF1 (Slave) · Mode 0
⚡ SPIMaster — Step by Step
1
Check o_busy=0 — safe to start. Driver waits in UVM TB.
2
Pulse i_tx_ready=1 with i_tx_byte=0x55. FSM latches into r_tx_byte.
3
PRE_COMM: r_ss←0 (SS goes LOW). First bit loaded to r_mosi from w_tx_byte[7].
4
COMM: r_cont_sclk counts → r_sclk toggles → r_edge_detect pulses each toggle.
5
Each SCLK falling edge: r_mosi ← w_tx_byte[7] (next bit). r_cycle_count++.
6
Each SCLK rising edge: r_rx_byte ← {r_rx_byte[6:0], i_miso}. Shift in slave bit.
7
r_cycle_count=8: FSM → POS_COMM. r_ss←1 (SS HIGH). o_mosi←Z.
8
IDLE transition: o_rx_byte←r_rx_byte (=0xF1). o_byte_ready pulses 1 cycle. o_busy←0.
📡 spiSlave — Step by Step
1
Preload TX buffer — i_tx_ready=1 with i_tx_byte=0xF1 while r_ss=1 (must be before SS falls).
2
SS falls: r_ss←0 (registered i_ss). o_busy←1. w_miso=w_tx_byte[7]=bit7 of 0xF1.
3
MISO drives: assign o_miso = r_ss ? 'z : w_miso. Now driving bit[7]=1 of 0xF1.
4
Each SCLK falling edge: r_cycle_count++ → w_tx_byte shifts → w_miso updates to next bit.
5
Each SCLK rising edge: r_rx_byte←{r_rx_byte[6:0], r_mosi}. Shift in master bit.
6
SS rises: r_ss←1. o_busy←0. o_miso←Z (releases bus). 8 bits collected in r_rx_byte.
7
r_ss=1 AND o_busy was 1: o_rx_byte←r_rx_byte (=0x55). o_byte_ready pulses 1 cycle.
8
Slave monitor captures o_rx_byte and o_byte_ready → sends transaction to scoreboard.
