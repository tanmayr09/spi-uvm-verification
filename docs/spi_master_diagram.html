🔄 Finite State Machine
Controls entire transfer lifecycle. Two always blocks:
always_ff updates state on clock edge  |  always_comb computes next_state
IDLE → PRE_COMM → COMM → POS_COMM → IDLE
Transition conditions: IDLE→PRE_COMM: i_tx_ready=1  |  COMM→POS_COMM: r_cycle_count≥8 && r_edge_detect  |  POS_COMM→IDLE: r_count_pos_com = DIVIDE_FREQ>>1
⏱ SCLK Generator
Counts FPGA clock edges to produce SCLK.

r_cont_sclk counts 0→DIVIDE_FREQ
r_sclk toggles when count hits limit
r_edge_detect pulses on each toggle

Formula: f_sclk = f_clk / 2(n+1)
Active only in STATE_COMM
📥 TX Shift Register
r_tx_byte latches i_tx_byte when
i_tx_ready=1 && state=IDLE

w_tx_byte = r_tx_byte << r_cycle_count
(shifts out MSB first for FRAME_FORMAT=0)

r_mosi ← w_tx_byte[7] on each SCLK edge
r_cycle_count increments each bit sent
📤 RX Shift Register
Samples i_miso on correct SCLK edge
(posedge for Mode 0)

r_rx_byte ← {r_rx_byte[6:0], i_miso}
shifts in LSB from MISO each clock

On IDLE: o_rx_byte ← r_rx_byte
o_byte_ready pulses for 1 cycle
➡ MOSI Driver
assign o_mosi = !r_ss ? r_mosi : 'z

When SS=LOW (active): drives r_mosi
When SS=HIGH (idle): HIGH-Z state

Prevents bus contention in multi-master
🕐 SCLK Output Logic
Adjusts r_sclk for CPOL setting:

CPOL=0: o_sclk = r_sclk (idles LOW)
CPOL=1: o_sclk = !r_sclk (idles HIGH)

Active during STATE_COMM + STATE_POS_COMM
🔴 SS Controller
r_ss register drives o_ss

PRE_COMM → r_ss=0 (assert SS LOW)
POS_COMM done → r_ss=1 (deassert)

If SS_PIN_ENABLE=0: o_ss = 'z
(user manages SS externally)
⚙ SPI Mode Select (CPOL / CPHA Decoder)
MODE 0: CPOL=0 CPHA=0 — sample rising, shift falling
MODE 1: CPOL=0 CPHA=1 — sample falling, shift rising
MODE 2: CPOL=1 CPHA=0 — sample falling, shift rising
MODE 3: CPOL=1 CPHA=1 — sample rising, shift falling
